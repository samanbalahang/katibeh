# Katibeh Naji WordPress Theme

A custom WordPress theme developed for the **Katibeh Naji** website.

This theme is built on top of the **Hello Elementor** theme and uses **Advanced Custom Fields (ACF)** to provide flexible content management and custom website functionality.

## 🌐 Live Demo

Demo Website:

https://samanbalahang.github.io/katibeh/

## 🚀 Features

- Built on **Hello Elementor**
- Powered by **Advanced Custom Fields (ACF)**
- Lightweight and optimized structure
- Custom templates and layouts
- Responsive design
- Easy content management through WordPress Admin
- Elementor compatible
- Developer-friendly codebase

---

## 📋 Requirements

Before using this theme, make sure your environment includes:

- WordPress 6.x or newer
- PHP 8.0+
- Elementor Plugin
- Advanced Custom Fields (ACF) Plugin
- Hello Elementor Theme (Parent Theme)

---

## 📦 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/USERNAME/REPOSITORY.git
```

### 2. Copy Theme Files

Move the theme folder into:

```text
wp-content/themes/
```

### 3. Install Required Plugins

Install and activate:

- Elementor
- Advanced Custom Fields (ACF)

### 4. Activate the Theme

Navigate to:

```text
WordPress Admin → Appearance → Themes
```

Then activate the Katibeh Naji theme.
